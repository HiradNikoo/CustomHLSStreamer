API Examples

Content Update (curl)

curl -X POST http://localhost:3000/api/update \
  -H "Content-Type: application/json" \
  -d '{"type":"content","data":"/path/to/video.mp4"}'

Layer Update (curl)

curl -X POST http://localhost:3000/api/update \
  -H "Content-Type: application/json" \
  -d '{"type":"layer","data":{"index":0,"path":"/overlay.png"}}'

Filter Command (curl)

curl -X POST http://localhost:3000/api/update \
  -H "Content-Type: application/json" \
  -d '{"type":"filter","data":{"command":"Parsed_overlay_1 x=100"}}'

Status Check (curl)

curl http://localhost:3000/api/status
